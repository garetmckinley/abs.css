abs.css is a minimal css framework for creating apps that utilize absolute positioning. It's completely possible to create a layout entirely by assigning class names to divs.

abs.css was created to be the grid system for obsxssion.com. Since both are in their infancy and being rapidly developed, it's not advisable to try to integrate abs into a production-level application. Not only is there no available documentation, but any class is subject to change at any moment.
